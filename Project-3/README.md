# Project 3 - Multilayer Switch Network with Router-based DHCP

## Technologies Used
- VLANs
- Trunking
- Multilayer Switching
- Switch Virtual Interfaces (SVIs)
- Inter-VLAN Routing
- Router-based DHCP
- DHCP Relay (IP Helper Address)
- Static Routing
- IPv4 Subnetting

## Network Overview
This project simulates a small enterprise network with four departments:
- HR (VLAN 10)
- Sales (VLAN 20)
- IT (VLAN 30)
- Management (VLAN 40)

A Multilayer Switch performs Inter-VLAN Routing, while a Router provides DHCP services for all VLANs. DHCP requests are forwarded using IP Helper Address. Static routing enables communication between the Router and the Multilayer Switch.
