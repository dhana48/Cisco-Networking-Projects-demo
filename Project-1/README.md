# Project 1 - Small Office Network

## Objective
Design a network connecting Accounts and Delivery departments using Cisco Packet Tracer.

## Technologies Used
- Cisco Packet Tracer
- Static Routing
- Subnetting
- IPv4 Addressing

## Devices Used
- 1 Router
- 2 Switches
- 4 PCs
- 2 Printers

## Features
- Two separate /29 networks
- Inter-department communication
- Proper IP addressing
- Successful ping connectivity
